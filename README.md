### EcoGuard: AI-Powered Indoor Monitoring & Automation

<img width="1895" height="937" alt="Web" src="https://github.com/user-attachments/assets/b55fdf02-4485-4874-b28d-1cdc2b670e26" />

An IoT system leveraging ESP32 and ThingsBoard for real-time environmental monitoring with on-device TinyML for Temperature/Humidity prediction. It features automated appliance control (fans/lights) based on sensor data and triggers intelligent email alerts when critical thresholds are breached.
